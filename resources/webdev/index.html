Web Development Resources

Front End

HTML

HTML (HyperText Markup Language) is the standard markup language for documents designed to be displayed in a web browser. It can be assisted by CSS and JavaScript.

  • W3Schools: A beginner tutorial for HTML, but it's more recommended to use it as a supplementary resource. W3Schools has a lot of examples, so it's easy for quickly looking up how to do something.
  • Khan Academy: Khan Academy's beginner webpages tutorial video, document, and interactive tutorials for both HTML and CSS.

CSS

Cascading Style Sheets (CSS) is a style sheet language used for stylizing the presentation of a document written in a markup language such as HTML. Stylization includes font, color, object size, and so on.

  • W3Schools: A beginner tutorial for CSS, but it's better used as a supplemental resource. W3Schools has a lot of examples, so it's easy for quickly looking up how to do something.
  • Khan Academy: Khan Academy's beginner webpages tutorial video, document, and interactive tutorials for both HTML and CSS.

JavaScript

JavaScript (JS) is a programming scripting language that can make a website interactive and dynamic. JavaScript can be used for both frontend and backend developing.

  • W3Schools: W3Schools supplies a JavaScript tutorial and many examples.
  • Khan Academy JS: An introduction to JavaScript, which is the prerequisite for the HTML DOM tutorial below, as well as the JQuery tutorial.
  • Khan Academy Interactivity: A tutorial for using JavaScript to make a webpage interactive using HTML Document Object Model (DOM).
  • W3Schools DOM: W3Schools also offers a JavaScript HTML Dom tutorial. However, it's more recommended to use Khan Academy's course to learn HTML DOM, and W3Schools to quickly search up certain methods.
  • Khan Academy jQuery: jQuery is a JavaScript library that simplifies HTML JavaScript DOM. Khan Academy has an interacvie video tutorial course on jQuery.
  • W3Schools jQuery: Again, W3Schools also has a tutorial, but it's better to use it to support the Khan Academy course.


Back End

PHP

PHP is one of the many server-side languages websites are built with. Many large websites and web apps, such as Yahoo, Facebook, Wikipedia, Wordpress, and Tumblr are developed PHP.

  • Laracasts: A video course that serves as an introduction to PHP.
  • PHP Enthusiast: A more in-depth tutorial.

Ruby (on Rails)

Ruby on Rails is a web application framework for Ruby, and is used to develop backend. Examples of websites and apps that use Ruby on Rails are Twitter, Hulu, GitHub, SlideShare, and Shopify.

  • Codecademy: A series of interactive tutorials.


Web Design

Usability/Accessibility

Usability defines how effectively and efficiently a user can use a technology. Accessibility makes it easier for those with special needs to access and navigate the site. Both are very important when designing web pages.

  • Free Code Camp: Offers a course specifically for accesibility in web development, along with responsive web design.
  • Learnux - Usability Course: Offers 21 videos (3 hours total) about design for usability.
